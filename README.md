# 🐍 Aprendiendo Python desde Cero

¡Hola! Este es mi repositorio personal para guardar todos los ejercicios, prácticas y apuntes del **Curso de Python 2025** de [midudev](https://youtube.com/@midudev).

El objetivo es aprender la sintaxis desde el "Hola Mundo" hasta conceptos más avanzados, enfocándome en entender bien el código y practicar mucho.

---

## 📺 El Curso
* **Vídeo en YouTube:** [Curso Completo de Python desde Cero](https://www.youtube.com/watch?v=TkN2i-_4N4g)
* **Repositorio oficial de Miguel:** [midudev/cursopython](https://github.com/midudev/cursopython)

---

## 📂 Contenido del Repositorio

El código está organizado por carpetas según los temas que voy viendo en el vídeo:

* `00_introduccion/` - Hola Mundo y la función `print()`.
* `01_variables/` - Tipos de datos, variables y uso de *snake_case*.
* `02_operadores/` - Operaciones matemáticas y lógicas.
* *(Iré añadiendo más carpetas a medida que avance en el curso)*

---

## 🛠️ Herramientas utilizadas
* **Python 3**
* **Visual Studio Code** (con la extensión de Python y *Pylance* activado)

---

## 🚀 Cómo ejecutar los archivos

1. Clona este repositorio en tu ordenador.
2. Abre la terminal en la carpeta del ejercicio que quieras probar.
3. Ejecuta el archivo con el comando:
```bash
   python nombre_del_archivo.py
```
